Практичні завдання

Завдання 1

Реалізуйте клас для перевірки, чи є рядок паліндромом (читається однаково в обох напрямках).

Завдання 2

Розробіть метод для конвертації температури між градусами Цельсія та Фаренгейта.

Завдання 3

Імплементуйте метод для обчислення факторіалу числа.

Завдання 4

Спроектуйте клас для генерації випадкових паролів із заданою довжиною.

Завдання 5

Розробіть об'єкт, який генерує градієнт між двома випадковими кольорами.

Завдання 6

Створіть метод для підрахунку кількості голосних і приголосних літер у рядку.

Завдання 7

Реалізуйте метод для перевірки, чи є число простим.
